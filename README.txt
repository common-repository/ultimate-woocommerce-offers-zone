== Description ==

This plugin lets the users to show sale products on the offer page moreover in the two different considerable ways. It is fully compatible with Enhanced WooCommerce Flash Sale that means WooCommerce offers Zone will start working automatically as soon as WooCommerce Flash Sale plugin is activated.


= Main Features =

1. **Set Number Of Sale Products** -The user can change the number of sale products to be shown on the offer page. This will help the users to ensure the right number of sale products at the right time. This is just to have eyes of all online shoppers attracted towards sales of products.

2. **Design Modification** - This extension mainly has two eye-catching designs that user can select according to needs. It can be changed anytime anywhere. This plugin provides to flexibility in concern with displaying offer zone.deals exactly in the way users want to show to their customers, delight them and ensure strong customer base as well.

3. **Heading Settings** - What is the first thing that arrests eyes of site visitor and online are the values-oriented headings that host lines below it? This feature of plugin lets the users to write their own headings according to them. In accordance with changes in the requirements, the plugin is fully customizable to make settings accordingly and make it feasible for users.

4. **Enhanced WooCommerce Flash Sale Plugin Compatibility** - This Extension “Ultimate WooCommerce Offers Zone” being compatible with [Enhanced WooCommerce Flash Sale plugin](https://wordpress.org/plugins/enhanced-woocommerce-flash-sale/) presents immense offers and sale together a time. Nothing can be best than having a view of sale! And offer! at the same time. The main purpose of this feature is to delight customers with the countdown of best deals plus sale in just a one approach.

5. **Single Click Activation** - This feature allows the user to activate the plugin in just a single click of the user. In just a single click, all its features would get activated to provide its benefits to users firmly.

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of Ultimate Woocommerce Offers Zone, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Ultimate Woocommerce Offers Zone" and click Search Plugins. Once you've found our Ultimate Woocommerce Offers Zone plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".


= Manual installation =

The manual installation method involves downloading our Ultimate Woocommerce Offers Zone and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).


= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
